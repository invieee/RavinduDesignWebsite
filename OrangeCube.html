Hares and Hyenas

Brand Design
3D Modelling
2022

The purpose of this project is to develop the front-end component of Hares and Hyenas Bookstore, an online platform for purchasing books. The implementation will involve the use of HTML, CSS and JS to create an interactive and visually appealing user interface.

The goals of this project are as follows:

a. Develop a user-friendly interface: Create a responsive and intuitive interface that allows users to easily navigate through the bookstore, view book details, and make purchases.

b. Enhance Usability: Improve user experience by implementing intuitive navigation and user-centric design principles. Focus on minimizing user effort, ensuring clear information communication, and incorporating user feedback to create a user-friendly interface.

c. Ensure Responsiveness: Ensure that the front-end component is compatible with differing screen sizes, providing a consistent experience for users across different platforms.
